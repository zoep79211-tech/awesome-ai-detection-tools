Crossplag's defining capability is cross-language plagiarism detection — the ability to identify when a text in one language has been translated from a source in another language. This is a meaningful gap that most plagiarism tools miss entirely, and it is particularly relevant in international academic institutions where students work across multiple languages.

The AI detection layer was added alongside this existing cross-language capability, creating a combined workflow: a single document submission can reveal AI-generated content, direct plagiarism within the same language, and plagiarism across language boundaries simultaneously.

The platform supports 100+ language pairs for the cross-language matching, with AI detection available for the major European languages. Detection quality on less common languages is lower, consistent with the industry pattern of models being most accurate on English.

Institutional deployment covers universities in over 50 countries, with a concentration in Europe and Latin America. The platform integrates with common LMS tools and offers batch submission for processing entire course sets at once.

The freemium tier allows a limited number of documents per month at no cost, which is sufficient for occasional use or evaluation. Institutional pricing is based on student headcount and covers unlimited submissions within the licensed period.
