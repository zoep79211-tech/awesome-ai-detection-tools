About AIToolsList

A community-maintained, open-source directory of AI tools. All data lives in plain Markdown files — no database, no ads, no affiliate links. Anyone can contribute via GitHub pull request.

What we track
🎯
Accuracy
Published or tested detection accuracy against known AI-generated text.
⚠️
False Positives
How often the tool incorrectly flags human text — critical for academic use.
💰
Pricing
Transparent tiers — Free, Freemium, Paid, Enterprise.
🔐
Signup Policy
Whether an account is required before you can use the tool.
🌍
Language Support
Tools tagged multilingual support detection beyond English.
⚡
API Access
Tools with an API can be embedded into your own workflows.
Data format

Each tool is a file at content/tools/slug.md with YAML front-matter. The JS reads an inline copy of this data so the site works without a server. 

---
title: 'My Tool'
slug: 'my-tool'
subtitle: 'One-line pitch'
website: https://mytool.com
price: Free
signup: false
accuracy: 96
false_pos: Low
tags: [free, academic, no-signup]
date: '2025-01-01'
---
