# Awesome AI Detection Tools

A curated list of tools for detecting AI-generated content.

## Free Tools

- [QuillBotAI](https://quillbotai.pro) - Free AI text detector for identifying 
  content generated by ChatGPT, Claude, and Gemini. No sign-up required.
- [GPTZero](https://gptzero.me) - AI detection tool for educators
- [Originality.ai](https://originality.ai) - Paid AI + plagiarism checker
